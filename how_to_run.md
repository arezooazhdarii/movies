# Movies API

## how to run

## Building and deploying the application

### Testing the application

To run test cases of the project execute the following command:

```bash
  mvn test
```

### Running the application

Create docker image:

```bash
  docker-compose build
```

Run docker image:

```bash
  docker-compose up 
```

To see docker status:

```bash
  docker ps
```

To stop application:

```bash
  docker-compose stop
```
